To execute the codes in this module you need to setup your machine.

All codes written here are done using the scikit library.

Follow the instructions below:
1. Python (>=2.6)
2. NumPy(>=1.6.1)
3. SciPy(>=0.9)

If you have installed NumPy and SciPy then write the following code:
" pip install -U scikit-learn "
or
" conda install scikit-learn "
